# Telegram Accounts Store

Минималистичная платформа для продажи Telegram аккаунтов.

## Структура
- `client/` - Публичный сайт для покупателей
- `admin/` - Админ-панель для управления объявлениями

## Особенности
- Минималистичный дизайн (белый/черный темы)
- Удобное меню навигации
- Автоматическое указание главного админа (@ваш_юз)
- Респонсив дизайн
- Быстрое развертывание на Vercel

## Развертывание
1. Создайте 2 проекта на Vercel
2. Загрузите `client/` и `admin/` в отдельные репозитории
3. Подключите к Vercel с настройкой доменов
